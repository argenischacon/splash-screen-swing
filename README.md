Un proyecto básico de Swing con pantalla de carga (Splash Screen), utilizando FlatLaf para una apariencia moderna.

## Tecnologías

*   **Java 21**: Lenguaje de programación.
*   **Maven**: Gestión de dependencias y construcción.
*   **FlatLaf**: Look and Feel moderno para aplicaciones Swing.
*   **SLF4J / Logback**: Framework de logging.

## Cómo empezar

1.  Clona el repositorio.
2.  Construye el proyecto con Maven:
    ```bash
    mvn clean install
    ```
3.  Ejecuta la aplicación.
